Place display images here and import them into your app from `src/assets/images`.

This app automatically loads local images from this folder using `import.meta.glob`.

Recommended file names for the card image keys:
- `caps.jpg`
- `toteBags.jpg`
- `businessCards.jpg`
- `plannersNotebooks.jpg`
- `tShirts.jpg`
- `jackets.jpg`
- `aprons.jpg`
- `tumblers.jpg`
- `pens.jpg`

If a local image is not present, the card will still fall back to the remote placeholder URL.
