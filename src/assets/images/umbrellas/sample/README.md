# Sample Images for Umbrellas Slideshow

This folder contains sample images that will be used in the product slideshow for the umbrellas category.

## Usage
- Place high-quality sample images here
- Images should showcase the umbrella products in various settings
- Supported formats: PNG, JPG, JPEG, WebP
- Images will be automatically loaded by the slideshow component

## Naming Convention
- Use descriptive names for the images
- Avoid special characters in filenames
- Keep filenames concise but meaningful
